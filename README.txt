--------------------------- Version Française --------------------------



<=====================> Pour lancer le programme <=====================>

- Installer java => 
https://www.java.com/fr/download/help/download_options.html

- Installer java SE =>
https://www.oracle.com/java/technologies/java-se-glance.html

- Lancer un terminal de commande et se placer dans le repertoire 
où il y a le fichier gypsy's_carnival.jar

- Exécuter la commande "java -jar gypsy's_carnival.jar"

- Jouer


<==========================> Pour jouer <=============================>

- Le jeu est entièrement en ligne de commande

- Vous avez plusieurs commandes qui ont chacune des spécificités

- Pour afficher les commandes tapez "help" et suivez les instructions

- Le but du jeu est de finir les 9 mini-jeux proposés par les
propriétaires de chaque stand du carnavale



--------------------------- English Version ---------------------------



<======================> To launch the program <======================>

- Install java => 
https://www.java.com/fr/download/help/download_options.html

- Install java SE =>
https://www.oracle.com/java/technologies/java-se-glance.html

- Launch a command terminal and move into the directory 
where there is the file gypsy's_carnival.jar

- Run command "java -jar gypsy's_carnival.jar".

- Play


<============================> To play <=============================> 

- The game is entirely on command line

- You have several orders, each of which has its own specificities

- To display the commands type "help" and follow the instructions.

- The goal of the game is to finish the 9 mini-games proposed by the
owners of each carnival stand
